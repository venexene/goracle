# Устройство Go

Самый глубокий слой — рантайм и компилятор Go. Планировщик, управление памятью, escape-анализ и сборщик мусора. Темы, которые отличают middle-разработчика от junior.

## Рекомендуемый порядок

1. **Планировщик Go** — понять, как рантайм управляет горутинами
2. **Память в Go** — разобраться, где живут данные
3. **Escape Analysis** — узнать, как компилятор распределяет память
4. **Сборщик мусора** — освоить автоматическое управление памятью в Go
