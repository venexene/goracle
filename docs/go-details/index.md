## Рекомендуемый порядок

1. **Типы данных** — освоить базовые типы и zero values
2. **Массивы и срезы** — понять фундаментальную структуру для последовательностей
3. **Мапы** — разобраться с хэш-таблицами Go
4. **Строки** — научиться работать с текстом правильно
5. **Интерфейсы** — освоить главный механизм абстракции
6. **Каналы** — перейти к конкурентной коммуникации
7. **Defer** — понять отложенное выполнение
8. **Замыкания** — освоить функции, хранящие состояние
9. **Ошибки** — научиться правильно обрабатывать ошибки
10. **Контекст** — управлять жизненным циклом операций
11. **Синхронизация** — освоить примитивы синхронизации
12. **Тестирование** — научиться писать тесты на Go без внешних библиотек
13. **Дженерики** — писать обобщённый код
14. **Базы данных в Go** — работать с БД через `database/sql`, `sqlx`, `pgx`
